Export Workout Data to CSV

This utility converts your historical workout data from the seed file into CSV format that you can import into Google Sheets.

Instructions:

  1. Click the button below to download your data as a CSV file
  2. Open your Google Sheet and add a new tab called "Workouts"
  3. File → Import → Upload the CSV file → Import into current sheet
  4. Done! Your historical data is now in Google Sheets
